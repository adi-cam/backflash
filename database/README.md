# Movie Database

## Entity Model

- Movie
    - id
    - title
    - genre [documentary, thriller, sci-fi, drama, comedy, romance, animation]
    - length (minutes)
    - country [ch, de, au, ...]
    - director
    - event_id

- Event
    - id
    - title
    - type [screening, lecture, discussion, guided_screening]
    - date
    - series
    - topic
